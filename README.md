超声波测距
=============================================

使用 Arduino 进行超声波测距的实验。使用 HC-SR04 模块进行超声波测距，使用 LCD1602 显示测量结果，并利用 DHT11 测量温度来计算出较为准确的声速。程序中使用 "Time" 库来使实现定时器中断的功能，[可以在这里下载 "Time" 库](http://srmonk.blogspot.com/2012/01/arduino-timer-library.html)
